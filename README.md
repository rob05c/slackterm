# slackterm
Slack terminal client written in Go

To run, put a text file named 'slack_token' in the path with your Slack token. To create a Slack token, go to https://api.slack.com/web#authentication and click 'Create token'. Then, copy the token text to a file named 'slack_token'. The file must contain the token exactlyâ€”no newlines or spaces.
